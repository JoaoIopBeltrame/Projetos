# Projetos

Repositório com projetos organizados por linguagem. Cada pasta representa uma linguagem e contém os projetos feitos nela, cada um com seu próprio README explicando o conteúdo.

## Organização

| Pasta | Linguagem | O que tem |
|---|---|---|
| [`C/`](./C) | C | Calculadora de Bits & Bitwise & Conversores de Bases |
| [`Python/`](./Python) | Python | Calculadora de Matrizes |

## Como navegar

Cada linguagem tem sua pasta. Dentro dela, cada projeto tem sua própria pasta com README explicando o que é, como rodar e as decisões técnicas específicas.
